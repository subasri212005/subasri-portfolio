# ✨ Subasri | Portfolio Website

Premium dark-themed responsive portfolio website showcasing my projects, skills, and contact info. Built with pure HTML, CSS & JavaScript.

🔗 **Live Demo:** [[subasri-portfolio.vercel.app](https://subasri-portfolio.vercel.app](https://subasri-portfolio-six.vercel.app/)) 

---

## 🚀 Features

- **Premium UI** - Glassmorphism + Neon gradient effects
- **100% Responsive** - Works perfectly on Mobile, Tablet & Desktop
- **Smooth Animations** - CSS keyframes + Hover effects
- **Contact Section** - Direct email, WhatsApp, LinkedIn, GitHub links
- **Fast Loading** - Pure HTML/CSS, No heavy frameworks
- **SEO Friendly** - Semantic HTML structure

---

## 🛠️ Tech Stack

| Technology | Usage |

## 📱 Sections

1. **Hero Section** - Name, role, CTA buttons
